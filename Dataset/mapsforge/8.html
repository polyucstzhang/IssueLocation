mapsforge commented Nov 6, 2014

From diewurst...@googlemail.com on October 03, 2010 12:12:16

Setup postgres server. 8.4 works for me. Use create_tablespace.sql in directory ./res to create tablespace and user if you have to. Use map_import.sql and osm_base.sql to create tables and whatever.

Use XML2PostgreSQL ./res/conf/preprocessing.properties berlin.osm 15 to populate database.
Use BinaryFileWriter ./res/conf/preprocessing.properties ./berlin.mf.map 1 ./res/conf/mapFileFeature.properties to generate berlin.mf.map file.

Have fun. Perhaps someone can comment on settings like base-zoom-level...

File for testing: http://page.mi.fu-berlin.de/pborgers/berlin.map
