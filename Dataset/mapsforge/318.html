From khaa...@gmail.com on May 25, 2012 08:13:48

Hi,
I don't know exactly how geofabrik updates their planet, but the files for the update are temporary at this place: http://planet.openstreetmap.org/redaction-period/ I'm using osmupdate( http://wiki.openstreetmap.org/wiki/Osmupdate ), for updating my osm-files, and it works quit good.
Example for the use with a planet.osm.pbf(with the temporary update url):
./osmupdate planet.osm.pbf planet_new.osm.pbf --planet-url= http://planet.openstreetmap.org/redaction-period/ --hourly
(I haven't tried that exactly, but it should work. The --hourly, because daily doesn't work, if I remember right)
