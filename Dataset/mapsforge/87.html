What steps will reproduce the problem? 1. Create and display a map with a route.
2. Periodically update/redraw the map (e.g. by changing a circles position in the map)
3. Zoom in and out using the GUI. What is the expected output? What do you see instead? After a while the route is no longer drawn correctly, but some extra lines (leading from and to the center of the map) are added. What version of the product are you using? On what operating system? Ubuntu 9.10, mapsforge-map-0.2.0.jar Please provide any additional information below. The (assumed) bug can be fixed by making two following two methods in class org.mapsforge.android.maps.MercatorProjection reentrant:

@Override
public Point toPixels(GeoPoint in, Point out) {

    Point returnPoint2 = null;

    // save the current position and zoom level of the map
    GeoPoint mapCenter2 = this.mapView.getMapCenter();
    byte mapZoomLevel2 = this.mapView.getZoomLevel();

    // calculate the pixel coordinates of the top left corner
    double pixelX2 = longitudeToPixelX(mapCenter2.getLongitude(), mapZoomLevel2)
            - (this.mapView.getWidth() >> 1);
    double pixelY2 = latitudeToPixelY(mapCenter2.getLatitude(), mapZoomLevel2)
            - (this.mapView.getHeight() >> 1);

    if (out == null) {
        // create a new point object
        returnPoint2 = new Point();
    } else {
        // reuse the existing point object
        returnPoint2 = out;
    }

    returnPoint2.x = (int) (longitudeToPixelX(in.getLongitude(), mapZoomLevel2) - pixelX2);
    returnPoint2.y = (int) (latitudeToPixelY(in.getLatitude(), mapZoomLevel2) - pixelY2);
    return returnPoint2;
}

@Override
public Point toPoint(GeoPoint in, Point out, byte zoom) {
    Point returnPoint3 = null;
    if (out == null) {
        // create a new point object
        returnPoint3 = new Point();
    } else {
        // reuse the existing point object
        returnPoint3 = out;
    }
    returnPoint3.x = (int) longitudeToPixelX(in.getLongitude(), zoom);
    returnPoint3.y = (int) latitudeToPixelY(in.getLatitude(), zoom);
    return returnPoint3;
}
