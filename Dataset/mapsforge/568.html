jordantrain commented Jan 14, 2015

My own MarkerShow class extends your Marker. I think this is the best candidate. It contains a call to Marker.contain which later makes a call in the Marker class to create a Rectangle.

        public class MarkerShow extends Marker{
            MarkerHide hideableMarker;

            public MarkerShow(LatLong wpt, Bitmap bmp, int horizontalDisp, int verticalDisp){
                super(wpt,bmp,horizontalDisp,verticalDisp);
            }

            @Override
            public boolean onTap(LatLong tapLatLong, Point viewPosition, Point tapPoint){
                //Toast.makeText(context, "Pin Tapped", Toast.LENGTH_LONG).show();
                Layers layers = mapView.getLayerManager().getLayers();
                if(hideableMarker != null && this.contains(viewPosition, tapPoint) && !layers.contains(hideableMarker)){
                    layers.add(hideableMarker);
                    mapView.getLayerManager().redrawLayers();
                }
                return false;
            }

            public void add(MarkerHide hideable){
                hideableMarker = hideable;
            }

        }
