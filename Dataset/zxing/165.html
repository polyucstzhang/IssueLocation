micjahn commented Jun 5, 2014

The difference between the two versions of zxing is the new ECI support for PDF417.
#27
The PDF417 encoder use the codepage Cp437 by default now if no other codepage is specified.
Before it uses the system default codepage I think.
But the specs of PDF417 says that Cp437 is the default codepage and all other codesets need an ECI segment.
A possible workaround for your devices: you can specify another codeset with EncodeHintType.CHARACTER_SET
